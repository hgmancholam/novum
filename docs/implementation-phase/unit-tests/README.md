# Unit Tests Documentation

This folder contains documentation about unit tests generated by the Coder Agent.

## Structure

- **backend/**: Backend test documentation and coverage reports
- **frontend/**: Frontend test documentation and coverage reports

## Test Requirements

### Backend (Python/pytest)
- All new functions must have unit tests
- Use pytest + pytest-asyncio for async code
- Mock external dependencies with pytest-httpx
- Target: >80% coverage

### Frontend (Vitest)
- Component tests with Testing Library
- Accessibility checks with jest-axe
- API mocking with MSW
- Target: >80% coverage
