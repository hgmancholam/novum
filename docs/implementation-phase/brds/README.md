# Business Requirements Documents (BRDs)

This folder contains Business Requirements Documents generated by the BSA Agent.

## Structure

- Each BRD follows the naming convention: `BRD-{YYYY-MM-DD}-{feature-name}.md`
- BRDs are generated from user requests and requirement analysis
- All BRDs must be reviewed before implementation begins

## Template

See `.github/memory-bank/templates/brd-template.md` for the standard BRD format.
